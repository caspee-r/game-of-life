
# Build
```console
$ make -B
```

# Controls

| Control                                   | Description                                      |
|-------------------------------------------|--------------------------------------------------|
| <kbd>space</kbd>                          | pause/play the simulation.                       |
| <kbd>q</kbd>                              | Quit the application.                            |
| <kbd>n</kbd>                              | generate the next generation (while paused).     |
